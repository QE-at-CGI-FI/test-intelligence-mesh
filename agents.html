Agents

Six practical agents for quality engineering — each with a use guide for GitHub Copilot and Claude Code. Start at L1. Build from there.

Skills are learned one at a time and practiced until they feel automatic. Combos are unlocked when you’re ready — not on a schedule. Chains become routines only once the individual moves are solid. You decide when you’re speed-ready.

L1 · Solo

A single move. Full value on its own. Practice until you can direct it precisely.

L2 · Combo

Works alone, but more powerful when paired with an L1 skill you’ve already practiced.

L3 · Chain

Designed for workflow integration. Most powerful when sequenced with other skills.

Two tools

GitHub Copilot and Claude Code

Every agent has instructions for both. They’re complementary — use whichever fits the moment.

◆

GitHub Copilot

Deeply integrated in VS Code. Great when you want context from your open editor, workspace files, or terminal history. Your AI pair programmer — always nearby.

◆

Claude Code

A CLI tool that takes a wider view. Reads multiple files, creates and edits code, and runs commands. Use it when you need an agent to act on the codebase, not just advise.
