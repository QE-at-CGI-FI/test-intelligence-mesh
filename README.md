# Test Intelligence Mesh

Connecting agents and skills, and leveled learning of AI for quality engineering purposes. While we have been experimenting capturing knowledge, we come to note that this is like building a mesh, one fiber at a time. Being able to scale requires humans in the learning loop, and before we can go fast, we need more control on the direction.
